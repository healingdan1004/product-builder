# 📝 LottoGen 블로그 포스팅 가이드

LottoGen 프로젝트에 새로운 블로그 포스트를 올리는 방법은 **자동 스크립트**와 **수동 작성** 두 가지가 있습니다.

---

## 🚀 1. 자동 스크립트로 포스팅하기 (권장)

가장 빠르고 간편한 방법입니다. 제목, 요약, 본문만 입력하면 파일 생성부터 메인 페이지 등록까지 자동으로 처리됩니다.

### 실행 방법
터미널에서 아래 명령어를 입력하세요:
```powershell
uv run scripts/add_post.py
```

### 입력 항목
1.  **제목 (한글/영문)**: 사이트에 표시될 제목입니다.
2.  **요약문 (한글/영문)**: 메인 페이지 카드에 표시될 짧은 설명입니다.
3.  **이미지 파일명**: `assets/blog/` 폴더에 넣어둔 이미지 파일명을 입력하세요 (예: `luck.jpg`). 없으면 그냥 엔터를 칩니다.
4.  **본문 내용 (Markdown)**: 마크다운 형식을 지원합니다. `#`으로 제목을 만들고, `**`로 강조할 수 있습니다. 입력이 끝나면 `Ctrl+Z` (Windows) 또는 `Ctrl+D` (Mac/Linux)를 누른 후 `Enter`를 치면 저장됩니다.

---

## 🖼️ 2. 이미지 올리는 법

블로그에 이미지를 넣으려면 먼저 파일을 아래 경로에 복사해야 합니다:
`assets/blog/`

*   **권장 사이즈**: 가로 1200px (Hero Image 스타일)
*   **파일 형식**: `.jpg`, `.png`, `.webp` 등

---

## 🛠️ 3. 수동으로 포스팅하기 (직접 수정)

스크립트를 쓰지 않고 직접 수정하고 싶을 때는 다음 단계를 따르세요.

1.  **파일 생성**: `blog/template.html`을 복사하여 `blog/새파일명.html`을 만듭니다.
2.  **내용 수정**: 생성한 파일 내부의 `{{title}}`, `{{content}}` 등을 실제 내용으로 바꿉니다.
3.  **메인 페이지 등록**: `index.html`의 `<div class="blog-grid">` 섹션 안에 새로운 `<article>` 블록을 추가합니다.
4.  **번역 등록**: `main.js`의 `translations` 객체에 새로운 키값을 추가합니다 (필수).
